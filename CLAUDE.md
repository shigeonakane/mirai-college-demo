# CLAUDE.md - 未来創造専門学校 デモサイト

## プロジェクト概要

CampusSite Partner の営業用デモサイト。日本の専門学校向けに販売するため、**日本風のデザイン**を採用する。

---

## デザインガイドライン

### デザインテイスト

**日本の専門学校ウェブサイト**らしいデザイン：
- 信頼感・安心感を重視
- 情報がしっかり伝わる構成
- 清潔感のある配色
- 過度な装飾は避け、読みやすさを優先

### カラーパレット

| 用途 | 色名 | カラーコード | 使用場面 |
|-----|------|-------------|---------|
| プライマリ | 信頼の青 | #2563EB | ヘッダー、CTAボタン、見出し |
| セカンダリ | アクティブオレンジ | #F97316 | CTAボタン（強調）、アクセント |
| アクセント | 成長の緑 | #10B981 | 成功メッセージ、実績表示 |
| 背景（明） | ライトグレー | #F3F4F6 | セクション背景 |
| 背景（白） | ホワイト | #FFFFFF | メイン背景 |
| テキスト（濃） | ダークグレー | #1F2937 | 本文 |
| テキスト（薄） | ミディアムグレー | #6B7280 | 補足テキスト |

### タイポグラフィ

**フォントファミリー**
```css
font-family: "Noto Sans JP", "Hiragino Kaku Gothic ProN", "Hiragino Sans", Meiryo, sans-serif;
```

**フォントサイズ**
| 要素 | PC | スマホ |
|-----|-----|-------|
| h1（ページタイトル） | 2.5rem (40px) | 1.875rem (30px) |
| h2（セクション見出し） | 2rem (32px) | 1.5rem (24px) |
| h3（小見出し） | 1.5rem (24px) | 1.25rem (20px) |
| 本文 | 1rem (16px) | 0.9375rem (15px) |
| 補足 | 0.875rem (14px) | 0.8125rem (13px) |

**行間**
- 見出し: 1.4〜1.5
- 本文: 1.8〜2.0（日本語は行間広めが読みやすい）

### レイアウト

**コンテナ幅**
- 最大幅: 1200px
- 左右パディング: 16px（スマホ）/ 24px（タブレット）/ 32px（PC）

**セクション間隔**
- PC: 80px〜100px
- スマホ: 48px〜64px

**グリッド**
- 基本は12カラムグリッド
- カード: 3カラム（PC）→ 1カラム（スマホ）

### ボタンスタイル

**プライマリボタン（CTA）**
```css
background: #F97316;
color: white;
padding: 16px 32px;
border-radius: 8px;
font-weight: 700;
box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
```

**セカンダリボタン**
```css
background: white;
color: #2563EB;
border: 2px solid #2563EB;
padding: 14px 30px;
border-radius: 8px;
font-weight: 700;
```

### 画像スタイル

- 角丸: 8px〜16px
- アスペクト比: 16:9（ヒーロー）、4:3（カード）、1:1（人物）
- オーバーレイ: 必要に応じて半透明の青（#2563EB / 50%）

### アニメーション

- 控えめで上品に
- ホバー: transform: scale(1.02), transition: 0.3s
- スクロール: フェードイン（必要最低限）
- 速度: 0.2s〜0.4s

---

## コンポーネント規約

### ヘッダー

- 固定ヘッダー（スクロール時に背景色変化）
- 高さ: 70px（PC）/ 60px（スマホ）
- 構成: ロゴ + ナビ + 資料請求ボタン
- スマホ: ハンバーガーメニュー

### フッター

- 背景: ダークグレー（#1F2937）
- 構成: ナビリンク + 学校情報 + SNS + コピーライト
- コピーライト: © 2024 未来創造専門学校 All Rights Reserved.

### カード

- 背景: 白
- 影: box-shadow: 0 4px 6px rgba(0, 0, 0, 0.05)
- 角丸: 12px
- ホバー: 影を強める + 少し浮き上がる

### セクション見出し

```html
<div class="text-center mb-12">
  <p class="text-primary-600 font-bold mb-2">SECTION TITLE</p>
  <h2 class="text-3xl font-bold text-gray-900">日本語の見出し</h2>
</div>
```

## 画像素材

### 使用可能な素材

- Unsplash（商用利用可）
- Pexels（商用利用可）
- Pixabay（商用利用可）

### 画像の要件

- 日本人または東アジア系の人物が望ましい
- 明るく清潔感のある雰囲気
- 学校・教育現場の雰囲気
- 解像度: 最低1920px幅

### 画像最適化

- 形式: WebP推奨（フォールバックとしてJPG）
- 品質: 80%
- 遅延読み込み: loading="lazy"
